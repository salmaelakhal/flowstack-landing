# üöÄ FlowStack Landing Page

FlowStack is a modern, interactive SaaS landing page designed to showcase the features, services, pricing, and dashboard of a fictional productivity platform. This project demonstrates advanced frontend skills, interactive UI components, animations, and responsive design.

---

##  Features

* Fully responsive landing page
* Dark / Light mode toggle
* Interactive dashboard with charts
* Animated sections and transitions
* Pricing calculator and interactive pricing plans
* Testimonials section
* Chatbot integration (interactive)
* Smooth scrolling navigation

---

##  Sections Covered

### Navigation & Hero

* Navbar with call-to-action buttons
* Hero section with CTA and demo links

### Services & Features

* Analytics insights
* AI-powered automation
* Enterprise-level security
* Team collaboration tools
* Integrations with third-party apps
* Priority support

### Dashboard Preview

* User activity tracking
* Revenue and task stats
* Performance over time charts
* Recent activity feed

### Pricing

* Starter, Professional, Enterprise plans
* Monthly and annual options
* Custom pricing calculator

### Testimonials

* Showcase real client feedback
* Highlight satisfaction and success stories
